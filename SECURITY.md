# Security Policy

## Reporting a vulnerability

Please **do not open a public issue** for security problems.

Report vulnerabilities privately via [GitHub private vulnerability reporting](https://github.com/danm421/foundry-planning/security/advisories/new)
(preferred) or by email to <danmueller20@gmail.com>. Include reproduction steps
and the affected route/component if known.

You should receive an acknowledgement within 72 hours. Please allow a
reasonable window for a fix to ship before any public disclosure.

## Scope

- This repository (application code, workflows, dependencies).
- The deployed application and its client portal.

Out of scope: denial-of-service volume testing, social engineering, and
third-party services (Clerk, Plaid, Stripe, Neon, Vercel) — report those to the
respective vendors.
